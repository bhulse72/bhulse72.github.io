NHL Analytics

MATLAB Data Analysis GUI

Overview

A MATLAB application that analyzes statistical trends across 1.2M+ data points from every recorded NHL game, surfacing insights on the impact of coaching changes, rule modifications, and equipment evolution on league-wide and team-specific performance.

Technical Details

Built with MATLAB AppDesigner to create an interactive GUI for efficient data navigation and user input. Visualizations were generated dynamically based on user-selected filters for team, season range, and statistical category.
