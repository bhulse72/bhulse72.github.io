LineupLab

React Python Algorithms

Overview

LineupLab is a web-based platform designed to help swim coaches manage team rosters, track individual swimmer performance, and strategically build lineups for dual meets. The platform features an adversarial lineup generator that simulates opposing team strategies to produce competitive, constraint-satisfying lineups in real time.

Technical Details

Built with a React front end using custom hooks, component-based architecture, and centralized state management. The lineup optimization engine is powered by a heuristic-driven adversarial algorithm written in Python that enforces dual-meet rules and constraints while maximizing projected team score.
